# dotnet new --help 显示出所有可用的项目模板
用VS2017创建Model-Controller-View的MVC项目后。通过工具栏上的运行下拉框可选择不同的运行与调试模式。

. 还可以选择项目名称直接通过控制台运行。此时，每个请求的Info日志会输出到控制台上。
